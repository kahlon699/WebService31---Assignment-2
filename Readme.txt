Step 1: Run the project file

Step 2: Select Binding Endpoint

Step 3: Enter String to be Processed

Step 4: Click Get Result Button

it will show the List of words, Upper Case string, Length of the string and Words Length as result

Note:1) String is parsed into words by using whitespaces(single white space or Multiple white space) as a Seperator.
     2) If no string is input, service raise fault exception which is handled by the client
     3) If Binding Endpoint is not inputed by the user, Form Prompt you to Enter the endpoint.
